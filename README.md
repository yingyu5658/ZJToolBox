有一些实用功能

Options:
  -V, --version  output the version number
  -a, --about    关于
  -l, --log      查看日志
  -h, --help     display help for command

Commands:
  mld [options]  Malody功能
  lbl [options]  卢布卢功能
  ts [options]   时间戳转换工具
  bvd [options]  B站视频解析下载
